# n8n Automation Workflows

A collection of professional automation workflows designed to optimize digital operations and content distribution.

## 🤖 Featured Workflow: Academic Content Distributor
- **Goal**: Automate post distribution from academic tools to LinkedIn & Telegram.
- **Tools Integrated**: n8n, LinkedIn API, Telegram Bot API, RSS Feeds.
- **Logic**: Filters relevant academic updates and formats them for multi-channel publishing.

## 🛠 How to Use
1. Import the `.json` files into your n8n instance.
2. Configure your own API credentials for the respective nodes.
